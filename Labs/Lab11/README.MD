Lab 11: Hyperparameter Tuning & AutoML

Overview: This lab focused on optimizing machine learning models through systematic Hyperparameter Tuning and exploring the capabilities of AutoML for streamlined model selection.

What I Learned: I practiced the difference between model parameters and hyperparameters, implementing Grid Search and Random Search to find the ideal settings for a RandomForestClassifier.

What I Did: I successfully tuned a baseline model, achieving a cross-validated accuracy of 94.29% using both search methods. I also implemented the AutoGluon library to automate the entire pipeline, including feature engineering and ensemble creation.

Challenges: The primary challenge was understanding the efficiency trade-off; while Grid Search is exhaustive and guaranteed to find the best combination in a set range, Random Search proved to be much faster for exploring larger, more complex parameter spaces.
