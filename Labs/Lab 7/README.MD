Lab 7: Advanced Model Evaluation

Overview: This lab focused on implementing sophisticated evaluation techniques to gain a deeper understanding of model reliability beyond basic accuracy scores.
What I Learned: I practiced generating and interpreting Confusion Matrices to identify Type I (False Positive) and Type II (False Negative) errors, as well as calculating Precision, Recall, and F1-Scores.
What I Did: I implemented 5-fold Cross-Validation, which resulted in an average CV score of 78.68%. This provided a much more robust estimate of the model's true performance compared to a single train-test split.
Challenges: A key challenge was the conceptual application of Precision vs. Recall. I analyzed real-world scenarios, such as autonomous braking systems and airport security, to determine which metric is more critical based on the "cost" of specific types of errors.
